# AI 스토리 영상 생성기 + 썸네일 추천

## 🎯 프로젝트 개요
- **이름**: AI 스토리 영상 생성기
- **목표**: 텍스트 스토리를 AI가 분석하여 장면별 이미지를 생성하고 YouTube 썸네일까지 자동 추천
- **상태**: ✅ 작동 중

## ✨ 전체 완성 기능

### ✅ 완벽한 워크플로우
1. **스토리 입력** - 텍스트로 스토리 입력
2. **AI 씬 분석** - Gemini API가 스토리를 장면별로 자동 분할
3. **장면 길이 자동 최적화** - 3~10초 범위에서 자동 조절
   - 중요한 장면: 7~10초
   - 짧은 전환 장면: 3~5초
   - 시청자가 지루하지 않도록 최적화
4. **씬 편집 기능** - 각 장면의 설명, 시각적 요소, 길이를 개별 수정 가능
5. **실시간 진행률 표시** - 분석 및 이미지 생성 진행 상황 실시간 확인
6. **이미지 생성** - 나노바나나 프로로 각 장면별 고품질 이미지 생성
7. **이미지 프리뷰** - 생성된 이미지를 바로 확인
8. **개별 편집** - 각 이미지를 개별적으로 수정 가능
9. **🆕 썸네일 자동 추천** - 3가지 스타일의 YouTube 썸네일 자동 생성
   - 임팩트 스타일 (강렬한 첫인상)
   - 스토리 중심 (감정적 연결)
   - 미니멀 디자인 (세련되고 전문적)
10. **다운로드** - 개별 또는 전체 ZIP 다운로드
11. **JSON 내보내기** - 프로젝트 전체 데이터 내보내기

## 📋 사용 방법

1. **스토리 입력**: 텍스트 영역에 전체 스토리 입력

6. **다운로드**: 개별 또는 전체 다운로드

## 📊 API 엔드포인트

### POST /api/analyze-scenes
스토리를 분석하여 씬 정보를 반환합니다. 각 씬은 3~10초 범위로 자동 최적화됩니다.

**요청**:
```json
{
  "story": "스토리 텍스트"
}
```

**응답**:
```json
{
  "success": true,
  "scenes": [
    {
      "index": 1,
      "description": "씬 설명",
      "visualElements": "시각적 요소",
      "duration": 6,
      "startTime": 0,
      "endTime": 6
    }
  ],
  "totalDuration": 60
}
```

### POST /api/generate-image
나노바나나 프로를 사용하여 이미지를 생성합니다.

**요청**:
```json
{
  "prompt": "이미지 생성 프롬프트",
  "model": "nano-banana-pro",
  "aspectRatio": "16:9"
}
```

**응답**:
```json
{
  "success": true,
  "imageUrl": "이미지 URL"
}
```
